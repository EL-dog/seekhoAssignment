Seekho Assignment

This is an Android application built to demonstrate modern app development practices, including RecyclerView usage, fragment animations, and state management using ViewModel and StateFlow.

Assumptions

The project is written in Kotlin.

It follows a single-activity architecture with multiple fragments.

Built using Gradle v9.1.0

Supports Android API level 21 and above.

Features

RecyclerView: Displays a scrollable list of items with smooth animations.

Fragment Transactions: Switch between different screens seamlessly.

Animations: Includes animations for RecyclerView items to enhance user experience.

Hilt + ViewModel + StateFlow: The app uses Hilt for dependency injection, ViewModel for UI state management, and StateFlow for reactive state updates in fragments.
This app is an offline first feature build using roomdb.

Known Limitations

Currently, the app does not have any major limitations.
